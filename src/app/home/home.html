运行说明
  1. 安装NodeJS v8.x LTS版本: https://nodejs.org
  2. 使用taobao npm镜像: npm config set registry https://registry.npm.taobao.org/
  3. 安装gulp-cli，执行命令行: npm install -g gulp-cli
  4. 安装项目依赖的npm包: 在当前项目根目录(angular1-webpack-es6-pcss-uirouter-eslint)执行命令: yarn
  5. 运行开发模式: gulp serve
  6. 运行Release版本: gulp serve:dist
  7. 构建Release版本: gulp

    配置文件说明
  1. conf/css-loaders.config.js -- 支持 SCSS-like 语法(precss)、Grid布局(lost)、CSSNext语法、autoprefixer等。
  2. conf/webpack-base.conf.js -- Webpack配置文件。 其中, “webpack.ProvidePlugin”配置$、jQuery等全局变量对应哪些模块； “resolve.alias”配置路径或模块的别名；“webpack.optimize.CommonsChunkPlugin”与“entry”，则定义打包时分成几块。
  3. conf/webpack-copy.conf.js -- Webpack copy files 的配置文件，针对无法找到依赖关系的文件。
  4. src/index.ejs -- 相当于index.html，详情“html-webpack-plugin”。
